# アプリケーション概要

Web制作の仕事専門の求人サイト。

# なぜ作成したのか
マルチログイン機能を使用したサイトをイチから実装したいと思ったのがきっかけです。
その中でも求人サイトを作成した理由については、以前に求人サイトの営業をしていた経験があり、管理画面を使用したことがあったので作成のイメージがしやすかったためです。
